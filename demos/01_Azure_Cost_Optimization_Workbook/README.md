# Demo 1: Azure Cost Visibility and Optimization

**Timebox:** 8 minutes

## What This Demo Shows

This demo walks through three places in the Azure portal that help teams understand spending and identify optimization opportunities:

1. Azure portal navigation to cost information.
2. Cost Analysis in Cost Management.
3. Cost Optimization workbook.

## What Attendees Should Learn

- How to access cost data in Azure.
- How to organize resources in cost reports.
- Where to find recommendations for cost optimization.

## How To Follow Along During The Meeting

Open the Azure portal and follow these steps as the demo is presented.

1. Sign in to the Azure portal.
2. Open the target subscription, or open a resource group if the discussion starts at a narrower scope.
3. In the left navigation, open **Cost Management**.
4. Select **Cost Analysis**.
5. Review the default spend view and note the current cost trend.
6. Change the grouping to **Resource group**, **Service**, or **Tag** to see how costs can be organized for reporting.
7. Apply one or two filters to narrow the view and connect cost to ownership or workload boundaries.
8. Select Advisor service and choose Workbooks.
9. Review the recommendation areas shown in the workbook, such as idle resources, right-sizing, reservations, savings plans, and Azure Hybrid Benefit.

## Demo Flow

### 1. Access Cost Data In Azure

Start in the Azure portal and show where cost information is accessed at the subscription or resource group scope. The main point is that cost data is already available in the portal and does not require a separate tool to begin analysis.

### 2. Organize Cost Reports In Cost Analysis

Use Cost Analysis to show spend over time, then change grouping and filtering so attendees can see how reports become more useful when aligned to resource groups, services, or tags. This is the point in the demo where ownership and accountability should be emphasized.

### 3. Find Cost Optimization Recommendations

Move to the Cost Optimization workbook and show where optimization signals are surfaced. Use this section to explain that the workbook helps turn spend visibility into follow-up actions.

## Suggested Talking Points

- Cost data should be easy to access for both technical and financial stakeholders.
- Cost reports become more actionable when resources are grouped consistently.
- Optimization recommendations are most useful when they are tied to a clear owner and next step.

## Meeting Outcome

By the end of the demo, attendees should know where to look for cost data, how to view cost by logical ownership boundaries, and where Azure surfaces optimization recommendations for follow-up.
